# 🤖 Automated Object Sorter using Computer Vision & Load Cell

An intelligent embedded system built using Raspberry Pi that automatically detects, classifies, and sorts objects based on visual features and weight.

## 🚀 Features

- 📷 Real-time object detection using camera
- 🧠 AI-based classification using Computer Vision (OpenCV / TensorFlow)
- ⚖️ Weight measurement using Load Cell + HX711
- 🔄 Automated sorting using servo/motor control
- ⚡ Fully autonomous system with minimal human intervention

## 🛠️ Tech Stack

- Python
- OpenCV
- TensorFlow / Keras
- Raspberry Pi GPIO
- HX711 Load Cell Module

## 🔌 Hardware Components

- Raspberry Pi
- Pi Camera 
- Load Cell
- HX711 Amplifier
- Servo Motors / Relay Module


## 🧠 Working Principle

1. Object is placed on platform
2. Camera captures image
3. model classifies object
4. Load cell measures weight
5. Decision logic combines both inputs
6. Object is sorted into respective bin
